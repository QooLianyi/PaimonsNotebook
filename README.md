派蒙笔记本是一个安卓平台下开源的原神工具，能够提供游戏相关的一些便捷服务，如：实时便笺、深渊记录查询、祈愿记录分析、多账号切换、丰富的可自定义样式的桌面组件、角色资料、武器资料、...

未来将会推出更多的功能！

## 下载使用

- 派蒙笔记本仅适用于Android 8(API 26)至更高版本的Android设备

#### 从Github获取

- [点击此处查看最新的版本](https://github.com/QooLianyi/PaimonsNotebook/releases/latest)
